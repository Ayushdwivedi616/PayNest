# PayNest 💰

PayNest is a personal finance management web application that helps users track expenses, manage budgets, and plan their finances smartly.

## Features
- User authentication (signup / login)
- Expense tracking
- Budget management
- Smart financial dashboard
- Payment categorization

## Tech Stack
Frontend:
- React
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- Supabase / PostgreSQL

## Project Structure

PayNest
│
├── frontend
├── backend
├── supabase_schema.sql
├── package.json
└── README.md

## Installation

Clone the repository

git clone https://github.com/Ayushdwivedi616/PayNest.git

Go to project folder

cd PayNest

Install dependencies

npm install

Run frontend

npm run dev --prefix frontend

Run backend

npm run dev --prefix backend
